Movie API

The objective of this project is to build a server-side component of a "movies" web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favourite movies.

Endpoints

The table below documents the endpoints in this API - including details of what URL endpoints to target, the data to send as parameters and the data to expect as responses.

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Return a list of ALL movies to the user /movies GET None A JSON array holding data about all movies in database
Return data on single movie by title /movies/:title GET None A JSON object holding data about a single movie, containing movie title, description, genre details, director details and image URL. Example:
{
"title":"The Shawshank Redemption",
"description":"Over the course of several years, two convicts form a friendship, seeking consolation and, eventually, redemption through basic compassion.",
"genre":
{
"name":"Drama",
"description":"In film and television, drama is a category of narrative fiction (or semi-fiction) intended to be more serious than humorous in tone."
},
"director":"Frank Derabont",
"imageURL":"https://media.themoviedb.org/t/p/w300_and_h450_bestv2/9cqNxx0GxF0bflZmeSMuL5tnGzr.jpg"
}
Return data on genre by name /movies/genre/:genreName GET None A JSON object holding data about a movie genre, including name and description:
{
"name": "Drama",
"description": "In film and television, drama is a category of narrative fiction (or semi-fiction) intended to be more serious than humorous in tone."
}
Return data on director by name /movies/director/:directorName GET None A JSON object holding data about a director, containing name, bio and birth year:
{
"name": "Steven Spielberg",
"Bio": "American film director, writer and producer. A major figure of the New Hollywood era and pioneer of the modern blockbuster, he is the most commercially successful director of all time. Spielberg is the recipient of various accolades, including three Academy Awards, a Kennedy Center honor, four Directors Guild of America Awards, two BAFTA Awards, a Cecil B. DeMille Award and an AFI Life Achievement Award.",
"Birth": "1946"
}
Add a new user /users POST A JSON object holding data about the user to add, structured like:
{
"name": "Rory",
"favouriteMovie": []
} 		A JSON object holding data about the user that was added, including an ID:
{
"name": "Rory",
"favouriteMovie": [],
"id": "3edcab41-5dc5-476d-b71a-de3a1052dcad"
}
Update user info /users/:id PUT A JSON object holding data about the user to update, structured like:
{
"name": "Oscar updated name"
"favouriteMovie":[]
} 		A JSON object holding data added to the user:
{
"id": 2,
"name": "Oscar updated name",
"favouriteMovie":[]
}
Add a movie to list of favourites /users/:id/:movieTitle POST None A text message indicating whether the movie was successfully added to user id
Remove a movie from list of favourites /users/:id/:movieTitle DELETE None A text message indicating whether the movie was successfully removed
Deregister existing user /users/:id DELETE None A text message indicating whether the user was successfully deleted
